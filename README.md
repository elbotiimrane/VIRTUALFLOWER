## ✨ Overview
An interactive web app that lets users send lifelike, animated virtual flowers with personalized messages.

---

## 🚀 Features

- 🌿 Lifelike animations (wind sway, breathing motion)
- 🌸 Unique flowers (procedural variation)
- 💡 Parallax depth effect (mouse interaction)
- 💎 Premium lighting & shadows
- 💌 Personalized messages
- 🎨 Customizable bouquets
