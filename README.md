# BloomNote Studio

A lightweight virtual flower website built with plain HTML, CSS, and JavaScript.

## Run it

```powershell
npm start
```

Then open `http://localhost:3000`.

## Features

- Build a custom bouquet by selecting flowers
- Edit wrap colors, accent colors, themes, and card styles
- Add a recipient, title, personal message, and signature
- Attach music with a direct audio file URL
- Share through a generated link or your email app

## Notes

- Link sharing works best once the site is deployed to a public host.
- The email button opens the default mail app with the bouquet link prefilled.
